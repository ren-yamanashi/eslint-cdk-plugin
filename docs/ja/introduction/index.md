# Introduction

[AWS CDK](https://aws.amazon.com/cdk/) 用の ESLint プラグインです。

このプラグインを使うことで、AWS CDK のコードを ESLint でチェックすることができます。

- 構文エラーを検出します。
- AWS CDK のベストプラクティス違反の発見。

ESLint エディタとの統合は、リアルタイムにコードをチェックするのに便利です。
